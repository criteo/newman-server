Newman-server allows you to run a Postman collection (using the newman engine) and output the result in either JSON, JUnit or HTML format.

More information on the Github repository
Once you've built your collection in Postman, you can export it as a JSON file. This is the file that will be runnable by newman and newman-server.
The data fie is an optional JSON file used to define how the value of your collection's variables for different iterations.

Output format

Once you've ran your collection in Newman with the output format selected to be JSON, you will receive the run summary. Add the summary to a JSON file and this is the file that will be used by newman to convert to HTML report.
